# Vim

Tailored for Neovim, but will still work in vanilla Vim.

## Installation

```bash
git clone https://github.com/liujoey/.vim.git ~/.vim
cd ~/.vim
make install    # installs vim-plug + sets up symlinks + installs plugins
```

Run `make update` to update plugins.
